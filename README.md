# Exploratory Data Analysis of IT Job Vacancies in Kazakhstan (2023)

## Описание проекта
Данный проект представляет собой **портфолионный Разведочный анализ данных** IT-вакансий, опубликованных в Казахстане на платформе hh в 2023 году в период с июня по июль месяц.

Цель проекта - продемонстрировать навыки анализа данных, визуализации и интерпретации результатов на примере рельного рынка IT-вакансий.

---

## Цели проекта
- Продемонстрировать навыки разведочной аналитики данных (EDA).
- Проанализировать распределение IT-вакансий по разным параметрам.
- Выявить основные закономерности и тренды публикации вакансий.
- Показать умение формулировать аналитические выводы и рекомендации.

---

## Описание данных
- **Страна:** Казахстан
- **Период:** Июнь-июль 2023 года
- **Область:** IT-вакансии
- **Тип данных:** Опубликованные вакансии
- **Формат:** CSV

Датасет содержит информацию о вакансиях, включая дату публикации, зарплату, город, требуемый опыт работы. Используется для анализа структурных характеристик рынка IT-вакансий.

## Ограничения
- Анализ охватывает только период июнь-июль 2023 года.
- Результаты не отражают годовую динамику рынка труда.
- IT специальности включают очень широкие категории.
- Узкий диапазон опыта работы.

## Ключевые выводы
- Рынок IT-вакансий демонстрирует выраженные доминирование определенных городов и вакансий.
- Положительная зависимость зарплаты от требуемым опытом работы.
